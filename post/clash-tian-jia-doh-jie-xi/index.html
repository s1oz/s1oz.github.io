为clash添加doh地址

在没有其他路由可以操作的情况下,只能从自己电脑下手了,先安装个clash
安装完成后该导入节点的导入
添加doh的过程很简单
在General中有Mixin的选项,点击齿轮,粘贴以下内容

mixin: # object
  dns:
    enable: true    
    enhanced-mode: redir-host
    listen: :53
    nameserver:
      - https://223.5.5.5/dns-query
      - https://doh.pub/dns-query
      - tls://dns.rubyfish.cn:853
    fallback:
      - tls://1.0.0.1:853
      - tls://8.8.4.4:853
      - https://doh.opendns.com/dns-query
