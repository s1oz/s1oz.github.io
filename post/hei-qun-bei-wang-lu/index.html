Active Backup for Business 激活

替换以下数据依次输入连接激活:
URL NAS的地址
PORT 默认5001端口
USER 用户名
PASS 密码
SERIALNUMBER 黑群SN

https://URL:PORT/webapi/auth.cgi?api=SYNO.API.Auth&method=Login&version=1&account=USER&passwd=PASS
https://URL:PORT/webapi/entry.cgi?api=SYNO.ActiveBackup.Activation&method=set&version=1&activated=true&serial_number="SERIALNUMBER"

检测是否激活

https://URL:PORT/webapi/entry.cgi?api=SYNO.ActiveBackup.Activation&method=get&version=1

命令

显示网卡显示真实MAC地址

dmesg | grep eth0

删除谷歌二次验证

rm /usr/syno/etc/preference/sheldon/google_authenticator

查看cpu核心数

cat /proc/cpuinfo |grep "core id" | wc -l

查询cpu线程数

cat /proc/cpuinfo |grep "processor" | wc -l

6.2.x修改root密码

sudo -i
cd /etc/ssh
chmod 755 sshd_config
vim /etc/ssh/sshd_config

#PermitRootLogin prohibit password修改为PermitRootLogin yes保存重启

synouser --setpw root 6666666666
