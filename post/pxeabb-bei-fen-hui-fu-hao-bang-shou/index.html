PXE+ABB 备份恢复好帮手

· 2019-08-15 · # dsm
阅读量 0

1.PXE跟ABB是啥

a. PXE:

预启动执行环境（Preboot eXecution Environment，PXE，也被称为预执行环境）提供了一种使用网络接口（Network Interface）启动计算机的机制。这种机制让计算机的启动可以不依赖本地数据存储设备（如硬盘）或本地已安装的操作系统。

B. ABB:

(Active Backup for Business)使您可从Synology NAS 的一个中央控制台管理所有VMware、Hyper-V、Windows 设备和文件服务器备份。快速可靠的恢复选项借助用于 Windows 设备、虚拟机和单个文件的即时还原选项，帮助您在面临灾难时仍可运行基本服务。

所以通过这两个搭配可以利用群辉来实现无盘启动安装系统跟恢复备份镜像

2.需要做啥准备工作

a. 群辉安装所需套件

套件中心安装Active Backup for Business Web Station PHP 7.0这三个套件

b. 下载所需文件

压缩包中含有支持uefi启动的pxe跟集成abb的pe的压缩包 这

c. 配置文件路径

在群辉-控制面板-共享文件夹-新增uefi文件夹

将压缩包解压后放置在此目录中

并将文件夹中的boot.ipxe.cfg中的

set nas_ip 192.168.2.101

ip改为自己群辉的ip

d. 建立虚拟主机

在Web Stasion中新增虚拟机 如图配置

端口默认6666
文件根目录下为刚刚新建的uefi文件夹
下方选择Nginx跟PHP7.0

e. 启用群辉TFTP服务

群晖-控制面板-TFTP

勾选启用TFTP服务
TFTP跟文件夹选择刚刚新增的uefi文件夹

f. 路由器TFTP指向群辉

以OP为例,在dnsmasq文件中添加以下代码

enable-tftp
dhcp-boot=ipxe.efi,,192.168.1.2.101

ipxe.efi为uefi启动文件
192.168.1.2.101为群辉ip

g. 设置bios

电脑连接网线
bios中开启pxe启动相关选项

并调整启动顺序
如卡pe无法进入,请检查bios是否开启安全启动,如有关闭即可正常启动
到此配置结束

3.启动后可以干嘛

a. 连接PXE服务器并启动到PE

上电!开机!

b. 利用ABB恢复之前备份的镜像

进入到PE后点击是,启用共享并连接到设备

然后该怎么做不用我说了吧🤣
桌面的Synology Recovery Tool文件夹的recovery.exe
就是群辉的恢复工具,双击打开

登陆后选择备份恢复即可

c. 利用WinNTStup安装系统

通过smb服务连接到群辉

加载群辉上的系统安装镜像
萝卜青菜各有索爱嘛
同理,如果PE上的ABB软件过于陈旧,可以下载了放置在群辉上,挂载后使用
